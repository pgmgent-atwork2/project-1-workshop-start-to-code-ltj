Introductie

Welkom bij ons project! In deze tutorial leggen we stap voor stap uit hoe we een quiz bouwen met HTML, CSS en JavaScript. Zelfs als je nog nooit hebt geprogrammeerd, kan je dit leren. Wie weet wordt je aan de hand van deze tutorial en quiz een top programmeur.

Wat is een quiz?

Een quiz is een spel waarbij je vragen moet beantwoorden. Elke vraag heeft meerdere antwoorden, maar slechts één is juist. Het doel is om zoveel mogelijk juiste antwoorden te geven.

Wat heb je nodig?

  • Een computer
  • Een browser (Chrome, Edge, Firefox)
  • Een beetje geduld

Stap 0: Visual Studio Code installeren

We gebruiken Visual Studio Code (VS Code) als code-editor. Een code-editor is een programma waarin je code schrijft en bestanden beheert.

VS Code kan je gratis downloaden via de officiële website van Microsoft. Na installatie kan je gemakkelijk HTML, CSS en JavaScript bestanden maken en bewerken.

  • Ga naar de website van Visual Studio Code
  • Download de juiste versie voor jouw computer (Windows, Mac of Linux) Versies
  • Installeer het programma

Stap 1: Project aanmaken en HTML bestand

Nadat je VS Code hebt geopend, maak je een nieuwe map voor je project.

Daarna maak je je eerste bestand:

  • Rechterklik in de map in VS Code
  • Klik op New File
  • Maak een bestand aan met de naam index.html

HTML staat voor HyperText Markup Language en is de standaard opmaaktaal die de structuur en inhoud van webpagina's bepaalt. Het gebruikt 'tags' om elementen zoals tekst, afbeeldingen en koppen te definiëren, die browsers omzetten in een visuele pagina. Het is geen programmeertaal, maar een structuurtaal.

Stap 2: HTML

HTML is de structuur van onze quiz. Het zorgt ervoor dat we tekst, knoppen en vragen op de pagina kunnen zetten. Zonder HTML zou er niets zichtbaar zijn.

Stap 3: CSS

CSS staat voor Cascading Style Sheets en is de codeertaal die de lay-out en vormgeving van een website bepaalt. Het is de visuele 'stijl' van een pagina. Met css kunnen we dus volgende zaken aanpassen:

  • kleuren aanpassen
  • lettertypes kiezen
  • de grootte van tekst veranderen
  • elementen positioneren op de pagina
  • je website mooier en overzichtelijker maken

Stap 4: JavaScript

JavaScript is een veelzijdige, lichtgewicht scripttaal die wordt gebruikt om webpagina's interactief en dynamisch te maken. Naast HTML en CSS is het een kerntechnologie van het moderne web, waarmee gedrag zoals animaties, formuliervalidatie en inhoudsupdates worden aangestuurd. Het is een prototype-gebaseerde, dynamische taal die zowel in browsers als via Node.js op servers draait. Zonder JavaScript zou de quiz niet reageren op klikken of antwoorden.

JavaScript importeren

Deze code importeert functies om vragen op te halen, houdt bij welke vraag je zit en wat je score is, en toont vervolgens een vraag in de browser door de HTML van een element te updaten.

JavaScript vragen ophalen

Deze functie haalt de moeilijkheidsgraad uit de URL, laadt de juiste vragen (easy of medium of hard) en toont de eerste vraag in de browser, pakt de eerste vraag en toont die met de bijbehorende antwoorden in de browser.

Controle antwoorden

Deze functie controleert of het gekozen antwoord juist is, geeft feedback (juist/fout), verhoogt de score bij een goed antwoord en gaat door naar de volgende vraag of toont de eindscore als alles klaar is.

Stap 5: Onze vragen (JSON data)

We gebruiken JSON om onze vragen op te slaan. Elke vraag bevat:

  • Een vraag
  • Meerdere antwoorden
  • Het juiste antwoord

Hierdoor kunnen we makkelijk nieuwe vragen toevoegen zonder de code te veranderen.

Stap 6: Hoe werkt het spel?

JavaScript zorgt ervoor dat de quiz werkt. Het doet het volgende:

  • Toont vragen één voor één
  • Controleert of een antwoord juist is
  • Houdt de score bij
  • Toont het eindresultaat

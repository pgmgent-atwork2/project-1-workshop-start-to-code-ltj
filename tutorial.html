Introductie

Welkom bij ons project! In deze tutorial leggen we stap voor stap uit hoe we een quiz bouwen met HTML, CSS en JavaScript. Zelfs als je nog nooit hebt geprogrammeerd, kan je dit leren.

Wat is een quiz?

Een quiz is een spel waarbij je vragen moet beantwoorden. Elke vraag heeft meerdere antwoorden, maar slechts één is juist. Het doel is om zoveel mogelijk juiste antwoorden te geven.

Wat heb je nodig?

  • Een computer
  • Een browser (Chrome, Edge, Firefox)
  • Een code-editor (bijvoorbeeld Visual Studio Code)
  • Een beetje geduld

Stap 1: HTML

HTML is de structuur van onze quiz. Het zorgt ervoor dat we tekst, knoppen en vragen op de pagina kunnen zetten. Zonder HTML zou er niets zichtbaar zijn.

Stap 2: CSS

CSS (Cascading Style Sheets) is de taal waarmee je de stijl en opmaak van een website bepaalt. Met CSS kan je bijvoorbeeld: kleuren aanpassen lettertypes kiezen de grootte van tekst veranderen elementen op de pagina plaatsen (links, midden, rechts) je website mooier en overzichtelijker maken

Stap 3: js

HTML is de structuur van onze quiz. Het zorgt ervoor dat we tekst, knoppen en vragen op de pagina kunnen zetten. Zonder HTML zou er niets zichtbaar zijn.

Stap 4: Onze vragen (JSON data)

We gebruiken JSON om onze vragen op te slaan. Elke vraag bevat:

  • Een vraag
  • Meerdere antwoorden
  • Het juiste antwoord

Hierdoor kunnen we makkelijk nieuwe vragen toevoegen zonder de code te veranderen.

Stap 5: Hoe werkt het spel?

JavaScript zorgt ervoor dat de quiz werkt. Het doet het volgende:

  • Toont vragen één voor één
  • Controleert of een antwoord juist is
  • Houdt de score bij
  • Toont het eindresultaat
